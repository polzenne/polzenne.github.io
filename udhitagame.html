新入り魔王

ジャンル アクションRPG
プレイ時間 5時間～15時間
対象プラットフォーム Windows
概要 最強の勇者を迎え撃つ魔王の時間遡行もの

魔王城に召喚された魔王が3日後にやってくる勇者を倒すために、時間を巻き戻したり住民と仲を深めたりするARPGです。
リアルタイムで時間が経過し、城内を歩き回る住民に話しかけたりプレゼントをして好感度を上げ、一緒に戦うなどしてダンジョンを攻略したりします。
時間が巻き戻るアイテムをいつ使うかが鍵になります。

前向き勇者

ジャンル アクションRPG
プレイ時間 1時間～2時間
対象プラットフォーム Windows
概要 方向転換ができない勇者の逆襲

魔王の呪いにより方向転換ができなくなった勇者が元に戻るために奮闘するARPGです。
方向転換ができないので目の前のオブジェクトしか調べることができませんし、後ろにいる敵に攻撃することもできません。
方向転換の有難みを知るRPG。

お怒り信長

ジャンル アクションRPG
プレイ時間 2時間～3時間
対象プラットフォーム Windows
概要 死後の世界で信長が光秀に復讐するゲーム

死後の世界で信長が光秀に復讐するARPGです。
横画面マスバトルで、技能を強化して強くなっていきます。
アクション性の求められるゲームです。

わちゃわちゃダンジョンズ

ジャンル オートバトルRPG
プレイ時間 3時間～10時間
対象プラットフォーム Windows
概要 パーティを組んで敵を討伐するオート戦闘ゲーム

ギルドのメンバーでパーティを作成し、ダンジョンに派遣して攻略していくオート戦闘ゲームです。
キャラクターを雇用したり、武器を集めたり、組み合わせを考えて出発することが大事です。
初のグループ製作ゲームで、Strayひろまさんとあたりメさんと製作しました。
